# java-socket-server-client-chat

A simple server-cient chat application java sockets

# Usage
Compile these two files with the command below:
```console
javac ServerMTS.java ClientMTS.java
```
run the server:
```console
java ServerMTS
```
run the clients:
```console
java ClientMTS
```
